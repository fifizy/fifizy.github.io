素书20160331：盈满则亏

发表于 2016-03-31 | 分类于 素书 |

昨晚有些失眠，断断续续睡了两觉，今早7点过就醒了。醒来刷朋友圈，得知一同事父亲走了的噩耗。她父亲中风多年，行动不便，这位同事昨天刚请假回老家，计划是4月份把父母接到北京来一起生活，怎料老天开了这么一个玩笑。

大家纷纷给这位同事发短信，“坚强”“有我们”“有事儿随时电话”，一大早北京便有人张罗大家要不要都凑凑份子表示点心意。我发了几句安慰的话，份子钱准备过两天葬礼时转过去。

愿逝者安息，生者节哀。

去公司路上和老孙聊到人生圆满这个话题，我说他算人生赢家，他忙否认说他的生活挺平的。我说你要是平，那我就是贫困了。

“你想想，你多年轻，就能有现在的成就。我在你这个岁数，……*&…………&%&……）”

然后就是很多关于他我已知的血泪史。

聊到上天是公平的，世事盈满则亏，也许你家庭不好，但是父母对你的教育比过很多人；也许你比不上很多人，但是遥远的故乡父母安康、一切有保障；有些人看起来风风光光，可是常有子欲养而亲不在之痛。

老孙愿意用财富换父母安康。

我愿父母永远快乐。

素书20160329：没毛病

发表于 2016-03-29 | 分类于 素书 |

上海出差，今晚有时间，约了阿绿一起吃顿饭聚聚，也是播客《惊风火扯》的主播之一。没有酒也畅谈甚欢，因为太久没见，能听到太多故事。反正我听故事听饱了，讲故事讲饿了。

和阿绿聊到感谢这两年的各种经历，遇到各种各样的人，听他们说话，看他们做事，和他们一起滚在红尘。对于生活的态度在今年都很巧地转变，穷尽全力地想做一个更好的自己。是啊，很快意识到自己浅薄的不得了，于是才真的知道学习的重要性，不停学习的重要性。

我说就在来上海之前， 我遇到了一个单方面“一见钟情”的人。阿绿很诧异，“这个词居然从你的狗嘴里吐出来”，是啊，她懂，在这之前都默认生活就这样了，无所谓了。可是那天心突然那么动了一下，还没感动到谁，先把自己感动的泪流满面。新搬的家里，躺在刚铺好的床上，那晚我嘬了两口酒，抽了三根烟。

结果很重要，但我也不那么在意结果了。我以为生活就这样默默地下去了，只是当知道自己还有救的时候，感谢那个特别的你。

和阿绿商量着说能不能把惊风火扯继续下去，说着说着又变成了不然先新开一个试试。名字暂定《我有一个朋友》，I have a friend.

朋友，你没有酒，你有故事。

素书160328：“正式”开始北漂生活：搬家啦

发表于 2016-03-28 | 分类于 素书 |

由于这样那样的原因（穷），从大望路搬到了垡头。这次怕麻烦，选择了自如寓，除了省心，其实内心也是对公寓式生活有一些小期待，渴盼着预见有趣的人，喝着酒，听他们的故事。

可以预见的是生活至少会有如下的变化：

  • ※正式开始北漂生活：从走路上班，到倒两趟地铁，耗时50-60分钟。
  • ※更多可以冷静的时间：大把的冷静时间，可以自在地听歌、看书、写东西、学英语。我每天在公司会呆到8点左右回家。可能是由于生活习惯的不同，之前每天回家以后，好像你不得不坐在客厅和大家一起看电视，或者看电视时突然变成了聊工作。好几次我尝试回家后把自己关在房间里看书，又会被问到“你在干嘛？”“出来看电视啊”“诶，那件事儿怎么样了(工作)？”……
  • ※每天“可能”有大把的机会锻炼：自如寓的公共生活区域有一个小健身房：两台跑步机，三辆动感单车。我正在努力给自己下决心：每天去跑半个小时。
  • ※更有机会见识到多元化的北京：垡头在北京东南角，四环外。挨着富人区华侨城。在华侨城的衬托下，这一大片的未拆迁区域显得生活那么地艰难。听说垡头很乱，想起了“东穷西贵南富北匪”的成都北门。也让自己清醒清醒：北京，未必是你以为的那个北京。

明天出差，回来慢慢归置。

呐，新生活开始啦！~

正三观，谈DSP广告效果：为什么要对着一个卖扳手要汽车？

发表于 2016-03-24 | 分类于 营销 |

因工作关系，经常和各区域性广告公司打交道。传统地广告公司迫不及待地想转型，网推公司迫不及待地寻找新的“噱头”，说白了都迫切地要寻找到一个新的增长点已备过冬之用。于是目光齐刷刷地投向一个新工具：程序化购买。区域市场习惯笼统地称之为DSP。

基于RTB（实时竞价）模式进行的程序化购买，依托大数据能够进行对针对某一类人群的广告展示机会进行购买,能够很大程度上更加合理地提高广告预算的合理性，尽量避免造成和传统展示广告一样的预算浪费。

但是，说到底，即使通过程序化购买地方式进行广告投放，于广告主而言，无非是多了另一种广告通路。其作用、属性与电视广告、报纸广告、灯箱广告和电梯广告是一致的!仅仅是增加了广告曝光地机会，或者说让曝光曝光的机会更加合理。

那么，为什么在“精准”这个旗号下，越来越多的广告主和广告公司产生一种诸如“这玩意儿特别屌”“这玩意儿一枪一个准”“这玩意儿让人看到了广告马上就要买房子”的错觉？
