素书20171223:冬至的几种过法

发表于 2017-12-23 | 分类于 素书 |

周六，晴。

广州人认为全世界冬至都吃汤圆

四川人认为全世界冬至都喝羊肉汤

东北人认为全世界冬至都吃饺子

长沙人第一次知道原来大家还过冬至

一个小段子。

昨天是冬至，同事点了饺子，中午打吃特吃就一顿，饺子这种东西在北方自带仪式感，反正大节小节吃它应该不会犯错。

今天一起床，同乡室友便嚷嚷说买了羊肉，昨天没吃到。她主厨，我帮厨，忙活了一下午晚上总算吃到了热腾腾的『自制简阳羊肉汤』，暖和。

冬至了，吹着冬风，总觉得有些事没解决，耿耿在怀。莫名有些烦躁。

素书20171212:近况

发表于 2017-12-12 | 分类于 素书 |

周二，晴冷。

终于在上周六完成了PMP认证考试。我考的时候自我感觉良好，甚至提前40分钟做完了题交卷。出考场后和我们组另一提前交卷的人交流，都觉得感觉还不错。直到小组其他成员陆续出来，个个面带愁容，我们俩心里开始没底了。

不过总的来说，终于度过了为期三个多月的煎熬学习备考的时光，边工作边学习确实不是想象中那么简单的。

这三个月荒废了很多东西，看书、资讯、娱乐、美食、周末，甚至不怎么有时间陪伴小5。好啦，结束啦，那你们都回来吧～

因为和小5把回成都提上了计划，所以最近在疯狂地密集地寻求工作机会。就连出差的时候也不忘了约一些朋友见面帮忙出主意。于我而言，回成都随便找一份能糊口的工作是没有问题的，但是目前的阶段对我整体的职业规划来讲，应该找一份能够稳定长达5-10年并具有上升空间的工作。又加上年关将至，难。

元旦我回成都，小5去江西参加婚礼，回来以后计划1月份带她出去小玩几天。

素书20171126：待处理的低端人口

发表于 2017-11-26 | 分类于 素书 |

周日，晴，冷…

周六原计划做题，起床看见小5都快要死了，想着陪陪她。今天把400道的题做了200道，还好，错了46道，我自己比较满意。周一再利用空隙把答案讲解仔细看了，下班再做50道。

值北京安全大扫荡/大扫除之际，小5提出想去成都工作。对于这个决定我当然是拥护的，这表示未来更近了点，我也得将此作为一个小规划，解决各个制约因素，逐步实现，挺困难，不过结果是好的。
